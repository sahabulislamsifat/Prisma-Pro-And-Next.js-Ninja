### Learning Approach

This repository follows a **build-by-learning** method:

1. Understand the concept (in `/docs`)
2. Study the schema and model design (in `/prisma`)
3. Build APIs and components (in `/src`)
4. Test, debug, and optimize queries
5. Implement real-world features
6. Deploy and scale your app
